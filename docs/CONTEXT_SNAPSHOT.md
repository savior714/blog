# CONTEXT_SNAPSHOT.md

## 1. Project Overview
- **Name:** Minimalist Diary Blog
- **Goal:** A clean, minimalist blog for a pediatrician-developer (overreacted.io style).
- **Phase:** Initial setup and deployment completed (v1.0 Ready).

## 2. Tech Stack
- **Framework:** Astro 5 (Server Mode)
- **Styling:** Tailwind CSS 4, @tailwindcss/typography
- **Writing Studio:** Keystatic CMS (`/keystatic`) & Local CLI Scaffolder (`npm run new-post`)
- **Hosting:** Vercel (Astro Adapter, fixed via `vercel.json`)
- **Skill Integration:** `skills` submodule integrated

## 3. Core Rules
- Adhere to `docs/AGENTS.md` guidelines as the top priority.
- Typography-first design for maximum readability.
- Every change is executed after approval of an `implementation_plan.md`.
