# 나의 개인 블로그

Next.js와 Tailwind CSS로 구축된 개인 기술 블로그입니다.

## 기술 스택

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Content**: MDX (Contentlayer)
- **Deployment**: Vercel

## 시작하기

```bash
npm run dev
```
