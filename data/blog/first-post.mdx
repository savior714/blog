## 안녕하세요! 👋

이 글은 Git에 푸시하면 자동으로 게시됩니다. **Tailwind Nextjs Starter Blog** 템플릿을 사용하여 만든 블로그입니다.

## 블로그 특징

이 블로그는 다음과 같은 기능을 제공합니다:

- ✅ **SEO 최적화**: 검색 엔진에 최적화된 구조
- ✅ **다크 모드**: 사용자 선호에 따른 테마 전환
- ✅ **MDX 지원**: 마크다운에서 React 컴포넌트 사용 가능
- ✅ **댓글 시스템**: Giscus를 통한 GitHub 기반 댓글
- ✅ **검색 기능**: 블로그 내 검색 지원

## 코드 예제

```javascript
const greeting = '안녕하세요!'

```

## 앞으로의 계획

이 블로그에서는 다음과 같은 주제들을 다룰 예정입니다:

1. 웹 개발 관련 팁
2. 프로젝트 경험 공유
3. 기술 학습 기록
