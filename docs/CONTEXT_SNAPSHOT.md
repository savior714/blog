# CONTEXT_SNAPSHOT.md

## 1. 프로젝트 개요
- **이름:** Minimalist Diary Blog
- **목표:** 소아과 의사이자 개발자의 기록을 위한 정갈하고 미니멀한 블로그 (overreacted.io 스타일)
- **단계(Phase):** 초기 구축 및 배포 설정 완료 (v1.0 Ready)

## 2. 기술 스택 (Tech Stack)
- **Framework:** Astro 5 (Server Mode)
- **Styling:** Tailwind CSS 4, @tailwindcss/typography
- **Writing Studio:** Keystatic CMS (`/keystatic`) & Local CLI Scaffolder (`npm run new-post`)
- **Hosting:** Vercel (@astrojs/vercel adapter, `vercel.json` fixed)
- **Skill Integration:** `skills` 서브모듈 통합 완료

## 3. 핵심 규칙 (Core Rules)
- `docs/AGENTS.md`의 지침을 최우선으로 준수.
- 가독성 최우선 디자인 (Typography-First).
- 모든 변경 사항은 `implementation_plan.md` 승인 후 실행.
